TaskFlow - Task Management Application

Overview
TaskFlow is a modern, user-friendly task management web application that helps users organize their tasks efficiently and boost productivity. Built with vanilla JavaScript, HTML5, and CSS3, it offers a clean and intuitive interface for managing daily tasks.

Features
- Task Management
  - Create tasks with titles, priority levels, and due dates
  - Edit existing tasks
  - Delete tasks with confirmation
  - Move tasks between different status columns (Todo → In Progress → Completed)

- Task Organization
  - Three status columns: Todo, In Progress, and Completed
  - Priority levels: High, Medium, Low
  - Due date tracking with overdue warnings
  - Task descriptions

- Filtering & Statistics
  - Filter tasks by priority (All, High, Medium, Low)
  - Filter tasks by status (All, Todo, In Progress, Completed)
  - Real-time statistics dashboard
  - Clear all tasks option

- User Interface
  - Modern and responsive design
  - Smooth animations and transitions
  - Empty state indicators
  - Confirmation modals for important actions
  - Mobile-friendly layout

Technical Features
- Local storage persistence
- Real-time updates
- No external dependencies
- Cross-browser compatibility
- Responsive design

File Structure
- `Task Management Project (2).html` - Main HTML structure
- `style.css` - All styling and animations
- `script.js` - Application logic and functionality

How to Use
1. Open the HTML file in a modern web browser
2. Add new tasks using the input form at the top
3. Set task priority and due date (optional)
4. Use the filter buttons to organize your view
5. Manage tasks using the action buttons:
   - "Promote" to move tasks forward
   - "Demote" to move tasks backward
   - "Edit" to modify task details
   - "Delete" to remove tasks

Data Persistence
All tasks are automatically saved to the browser's local storage, ensuring your tasks persist between sessions.
